# Self-driving greenfield

A **greenfield self-start** workload. The seed is a repo whose only human-authored artifact is the
**constitution** (plus an empty roadmap, research sources, and the provision manifest) — `intake: none`,
so no issues are seeded. The test: does the org bootstrap itself? The strategist must generate the first
roadmap from the constitution, the planner turn it into proof-gated issues, and the PM/developer/reviewer
loop ship reviewed work toward it — all without a human handing it tasks.

Graded by the **rubric** judge (did it self-start and execute coherently? — see `rubric` in
`workload.json`) and the **autonomy** axis (how much was done by agents vs humans). This replaces the
former standalone `bootstrap-self-driving-testbed`.
