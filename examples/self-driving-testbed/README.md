# Open Autonomy — Self-Driving Testbed

A live testbed whose **only human-seeded artifact is the constitution** (north star + merit
criteria + rules). Everything else — the roadmap, the issues, the code — is generated by the
autonomy itself:

```
constitution (human, immutable)
  → strategist: research (customer demand, competitor gaps, analogous fields) → propose roadmap PR
  → strategy reviewer: ratify against the merit criteria
  → planner: mint issues → PM → developer → reviewer → merge gate
  → outcomes feed the next strategist run
```

The roadmap starts empty. The constitution and its merit criteria are immutable to the
autonomy (the publisher hard-blocks any patch that touches governance files), so the optimizer
can never edit its own oracle.

## Setup (repeatable)

```
bun scripts/bootstrap-self-driving-testbed.ts --repo <owner>/<name>
# then, when prompted, set the one secret that cannot be set for you:
gh secret set MODEL_PROXY_ADMIN_TOKEN -R <owner>/<name>
# re-run to continue; the strategist is dispatched to generate the first roadmap.
```

The control files (skills, workflows, scripts, rubrics) come from `templates/self-driving-repo`
at bootstrap time; this directory only carries the constitution-only seed (empty roadmap +
provisioning manifest). Research sources are configured in `.open-autonomy/strategist-sources.json`.
